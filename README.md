# Documentação do Modelo de Dados: Esquema Professor
* Visão Geral
* Este repositório contém o arquivo starschemadiagramaprofessor.mwb, que representa a modelagem de um banco de dados relacional.
* O foco do projeto é a organização de informações acadêmicas dentro de um esquema denominado professor.
* Estrutura do Projeto
* O arquivo foi construído utilizando a ferramenta MySQL Workbench .
* Modelagem Relacional: Criação de tabelas e definição de relacionamentos para o contexto de professores e possivelmente disciplinas ou departamentos.
* Desenvolvimento Visual: Uso do EER (Enhanced Entity-Relationship) para facilitar a visualização da integridade referencial do banco.4.
* Tecnologias Utilizadas :
* MySQL Workbench: Ferramenta principal de modelagem.
* XML: Formato de estruturação do diagrama.
* SQL: Linguagem base para a futura exportação do script DDL (Data Definition Language).
